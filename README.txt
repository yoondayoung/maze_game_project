* 프로젝트 실행 방법
1. 실행시키고자 하는 프로젝트의 bin/data에 압축파일 안 data폴더 안 "rank.txt" 파일을 넣는다.
굳이 "rank.txt"파일이 존재 하지 않아도 되지만, 기존 랭킹을 적용시키고 싶다면 랭킹 파일의 이름은 반드시 "rank.txt"여야 적용가능하다.
2. 실행시키고자 하는 프로젝트의 src폴더에 압축파일 안 src폴더의 파일들을 모두 넣는다.
3. openframeworks projectGenerator를 통해 프로젝트를 생성한 후 Microsoft Visual Studio에서 열고, 2.에서 추가한 파일들이 모두 src안에 포함되지 않았다면 파일을 열어 추가한다.
4. 활성 솔루션 구성은 Debug, 활성 솔루션 플랫폼은 x64로 설정한다.
5. 빌드 후 실행한다.

* 게임 실행 방법
1. Start 메뉴에서 게임모드를 선택하면 게임이 바로 실행된다.
2. 맨 왼쪽 윗칸이 출발 지점이고, 맨 오른쪽 아랫칸이 도착 지점이다.
3. 연보라색 원이 플레이어의 현재 위치를 나타내며, 방향키를 사용하여 위치를 조작할 수 있다.
4. 별이 있는 위치로 가서 별을 획득하면 star bonus 점수를 얻는다.
5. 도착지점에 도달하면 게임이 바로 끝나면 alert창을 통해 획득한 점수를 확인할 수 있다.
6. alert창을 마우스로 닫고 나면 바로 뜨는 textbox창에 이름을 입력하면 그 이름으로 랭킹에 점수가 저장된다.
   (이름을 저장하지 않을 시 default 이름은 Player이다.)

* 기타 메뉴
1. Start 메뉴에서 exit를 선택하면 프로그램이 종료된다.
   (exit를 클릭하여 프로그램을 종료한 경우에만 프로그램 실행 중에 추가된 랭킹정보가 rank.txt에 추가 및 새 rank.txt가 생성된다.)
2. View 메뉴에서 Show Answer를 선택하면 가장 최근의 미로의 정답 경로가 표시된다.
3. View 메뉴에서 Show Ranking을 선택하면 기존 rank.txt 파일에 있던 랭킹 정보 + 프로그램이 실행되고 난 후 추가된 랭킹 정보를 확인할 수 있다.
4. View 메뉴에서 Full screen을 선택하면 전체화면으로 윈도우 창을 볼 수 있다.
